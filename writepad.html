WritePad Editor

WritePad is an editor application that allow you to creat Web pages easily and quickly.
Step to open WritePad application:
  1. Click Start
  2. Click Program
  3. Click WritePad
Advantages of using a WritePad Editor:
  • It allows faster creation of Web pages.
  • It allows easier updation of text or image in WritePad editor.
  • It is flatform independent.
Disadvantages of using a WritePad Editor:
  • It consumes more memory space
  • It requires to fill-in more detail such as meta data for each page
